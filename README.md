  Tabelas de Cursos

Este projeto é um exercício acadêmico desenvolvido para praticar conceitos básicos de HTML e CSS, especialmente o uso de tabelas, alinhamento e estilização com classes.

 Sobre o projeto

A página apresenta três tabelas com cursos de diferentes áreas:

Área de TI
Área da Saúde
Área de Design
Cada tabela contém:
Nome do curso
Quantidade de disciplinas
Número de semestres
As tabelas são organizadas lado a lado utilizando Flexbox.

   Tecnologias utilizadas
Tecnologia	Descrição
HTML5	Estrutura da página e das tabelas
CSS3	Estilização, cores e alinhamento

   Objetivos do exercício:
Aplicar estruturação de tabelas no HTML
Utilizar classes para estilização no CSS
Organizar conteúdo com Flexbox
Trabalhar cores e espaçamentos via CSS

   Arquivo do projeto:
index.html — Contém todo o código HTML e CSS da atividade

   Como visualizar:
Baixe ou copie o arquivo index.html
Abra-o em qualquer navegador (Chrome, Firefox, etc.)
A página será exibida com as tabelas alinhadas no centro
